# Quiet Minds

{% hint style="info" %}
An open-source collection of calm, screen-safe educational games for children aged 3–7.
No ads · No sounds · No timers · No game over
{% endhint %}

Most apps made for young children are designed to be addictive, not educational.
They flash, they beep, they reward every tap with fireworks and fanfare.
A four-year-old sitting with a tablet shouldn't feel like they're in a casino.

We wanted to build something different: a quiet place where a child can think.
One game. One question. One answer. No noise, no surprises, no pressure.

## Design Principles

Every Quiet Minds game is built on three non-negotiable commitments:

* **[No audio — ever](design/decision-no-audio-d1a2.md)**
* **[No flashing or distracting animation](design/decision-no-animation-d2b3.md)**
* **[Accessible color palette — works for colorblind children](design/decision-accessible-palette-d3c4.md)**

## For Parents & Educators

* **[Why is there no sound?](about/faq-why-no-sound-f1a2.md)** — Quiet Minds has no sounds at all — no music, no beeps, no voices.
* **[What age is Quiet Minds for?](about/faq-what-age-f2b3.md)** — Quiet Minds is designed for children aged 3 to 7, but there are no rules.
